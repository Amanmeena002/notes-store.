# 📚 Notes Selling Website

A full-stack web application to upload, sell, and download notes online with secure payment and admin approval system.

---

## 🚀 Features

- 🔐 User Login System (JWT आधारित)
- 📤 Admin Panel (Notes Upload)
- 💰 Razorpay Payment Integration
- 📸 Payment Screenshot Upload
- ✅ Admin Approval System
- 🔒 Secure PDF Download (Only Paid Users)
- 📱 Mobile Friendly UI
- 📊 Basic Sales Dashboard

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express  
- Database: MongoDB  
- Payment Gateway: Razorpay
